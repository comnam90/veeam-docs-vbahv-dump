# Creating Backup Jobs


To create a backup job using the Veeam Backup & Replication console, do the following:

1. [Check prerequisites and limitations](backup_job_prerequisites.md).
2. [Launch the Add Job wizard](backup_job_vbr_launch_wizard.md).
3. [Configure general settings](backup_job_vbr_general_settings.md).
4. [Select resources to back up](backup_job_vbr_assign_vms.md).
5. [Configure backup target settings](backup_job_vbr_destination.md).
6. [Enable guest processing](backup_job_vbr_guest_processing.md).
7. [Create a schedule for the backup job](backup_job_vbr_schedule.md).
8. [Finish working with the wizard](backup_job_vbr_review.md).
